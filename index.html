Pydataexplorer

A file browser, implemented in python, designed to interactively plot delimited data files in a wxpython GUI interface

PyDataExplorer introduction

PyDataExplorer is intended as a simple, easy-to-modify program for interactive exploration of delimited text data files, such as those commonly produced during various scientific measurements. It is designed as a GUI file browser with additional functionality to allow realtime data previews in the form of matplotlib plots and spreadsheet-style grid views.

In addition, each of the components is designed for modular use within your own code. As an example of one such implementation, a custom "open file" dialog is presented, which pops up the the file browser GUI to allow the user to select a file to open. When looking through files to open in this popup window, the user sees previews of the data within the files right on the open file dialog, making it easier to ensure that the user is correctly selecting the right data file.

Dependencies

The following packages are needed in order to run this PyDataExplorer:

  • wxpython
  • matplotlib
  • numpy
  • pywin32 - only necessary on Windows computers. This library is provided to allow the Data Explorer the ability to display all possible drive letters on the computer.

Running PyDataExplorer

Two simple standalone app programs are provided, and can be run from the command line:

To run the standalone Data Explorer, run:

$ python DataExplorer.py

To see a simple demo of the Data Explorer used as a file dialog, run:

$ python ExplorerDialogDemo.py
